# Что загружать в репозиторий

## Обязательно

- исходный код (`src/`)
- `CMakeLists.txt`
- `README.md`
- папка `docs/`
- отчет (отдельным файлом при необходимости, например `report.pdf`)
- скриншоты работы в `docs/screenshots/`

## Не загружать

- `.vs/`
- `out/`, `build/`
- временные файлы и бинарники (`.exe`, `.pdb`, `.obj`, `.log`)

Это уже учтено в `.gitignore`.
